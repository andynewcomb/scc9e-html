13.11 Random numbers. If you ask a computer to generate “random numbers” between 0 and 1, you will get observations from a uniform distribution. Figure 13.12 shows the density curve for a uniform distribution. This curve takes the constant value 1 between 0 and 1 and is zero outside that range. Use this density curve to answer these questions.

  1. (a) Why is the total area under the curve equal to 1?

  2. (b) The curve is symmetric. What is the value of the mean and median?

  3. (c) What percentage of the observations lie between 0 and 0.1?

  4. (d) What percentage of the observations lie between 0.6 and 0.9?
