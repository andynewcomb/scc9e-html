Hand: Worthless One pair
Probability: 0.50 0.42
Hand: Two pairs Better hands

  2. (b) What is the expected number of hands a player is dealt before the first hand better than one pair appears? Explain how you would use simulation to answer this question, then simulate just two repetitions. (Hint: See page 472.)
